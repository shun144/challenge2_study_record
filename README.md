# サービス名

JISOU 学習課題の学習時間管理アプリ

# サービスの説明

- 作業名と時間を入力し学習時間を一覧管理できる React アプリケーションです。
- 登録した作業は削除可能。
- 作業時間の累計も表示。

# 環境設定の方法(.env など)

- フロントエンド：React/Javascript
- データベース：supabase
- CI/CD：Firebase/github actions
- テストツール：Jest/Testing-Library

# 起動の仕方

以下 URL にアクセスすると表示可能。

# 本課題で学習したこと

- JavaScript と React の基本
- supabase（BaaS）の DB 作成
- Firebase へのデプロイ
- github actions による CI/CD
- Jest と Testing-Library による自動テスト
